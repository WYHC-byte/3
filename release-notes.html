Overview

Updates to the TyloAI developer platform, including TyloAI API, clients, and ode-code CLI.

On this page:

  • Release notes for TyloAI clients.
  • Update notes for ode-code CLI.
  • For more ode-code CLI updates, please refer to the full CHANGELOG.md in the repository.

November 2025 New

TyloAI ode-7 Family Launch

We officially launched the TyloAI ode-7 model family, marking a new era of reasoning capabilities.

  • ode-7-reasoning: Our flagship model with "Post-Thinking" capabilities for complex tasks.
  • ode-7: The balanced core model for enterprise applications.
  • ode-7-flash: High-speed model supporting Extended Thinking.
  • Introduced -thinking and -post-thinking API suffixes for granular control over cognitive processes.

ode-code CLI v2.1

  • Added support for the new ode-7 model family.
  • New /init command to intelligently analyze workspace and generate project-specific instructions.

October 2025

Extended Thinking Beta

Opened public beta for Extended Thinking capabilities on the web interface.

  • Users can now manually toggle "Enable Extended Thinking" in the chat settings.
  • Added visualization for the thinking process in the UI (expandable logic blocks).

Web Search Integration Update

Improved the RAG pipeline for web search, reducing latency by 30% and improving source citation accuracy.

September 2025

ode-5 Deprecation Notice Important

Officially announced the End of Life (EOL) for the ode-5 model family.

  • All ode-5 API endpoints now return 410 Gone.
  • Documentation updated to reflect migration paths to ode-6 and ode-7.

ode-code CLI v2.0

Major version update for our command line interface.

  • Multi-file Editing: ode-code can now safely edit up to 20 files simultaneously with atomic commits.
  • Terminal UI Overhaul: Completely redesigned the interactive terminal interface for better readability.

August 2025

Artifacts General Availability

Artifacts moved from experimental to General Availability (GA) for all users, including the Free tier.

  • Added support for React, Vue, and plain HTML5 previews.
  • Introduced "One-Click Debug" console within the Artifact preview window.

July 2025

Prompt Caching Feature

Introduced Prompt Caching to significantly reduce costs for repetitive contexts.

  • Cache system prompts and large documents with cache_control: {"type": "ephemeral"}.
  • Up to 90% cost reduction for cached input tokens.

June 2025

Vision Capabilities Upgrade

Enhanced visual processing for the ode-6 family.

  • Improved OCR accuracy for handwritten text.
  • Better understanding of complex charts and scientific diagrams.

May 2025

ode-6 Maintenance Mode

With the upcoming ode-7 release, the ode-6 family has entered maintenance mode. No new features will be added, but security patches will continue until March 2026.

April 2025

Mobile App Updates

Released updated iOS and Android applications.

  • Sync chat history across Web and Mobile in real-time.
  • Added voice mode for hands-free interaction.

March 2025

New Regions

TyloAI API is now available in new European regions with reduced latency for EU customers.

  • Frankfurt (eu-central-1)
  • Dublin (eu-west-1)

February 2025

Python SDK v1.5

Released version 1.5 of the official TyloAI Python SDK.

  • Added native support for asynchronous streaming.
  • Improved type hinting for better IDE autocomplete support.

January 2025

Project "Ode" Teaser

First public announcement of the "Ode" architecture, promising a shift from pure prediction to reasoning-based AI models.
